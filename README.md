# MATE
## Installation

1. Install [uv](https://github.com/astral-sh/uv):

    ```bash
    pip install uv
    or
    pip3 install uv
    ```

4. Install or sync dependencies with `uv sync`:

    ```bash
    uv sync
    ```

3. Activate the environment:

    - On Unix/macOS:

      ```bash
      source .venv/bin/activate
      ```

    - On Windows:

      ```cmd
      .venv\Scripts\activate
      ```



You're now ready to start using MATE!
